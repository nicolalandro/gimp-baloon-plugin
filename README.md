# Add baloon plugin
This is a gimp plugin to create manga baloon easy.

* select an area
![select area](https://github.com/nicolalandro/gimp-baloon-plugin/raw/master/imgs/demo1.png)

* go to select Add baloon...
![add baloon](https://github.com/nicolalandro/gimp-baloon-plugin/raw/master/imgs/demo2.png)

* insert text and specify font and font size
![insert text](https://github.com/nicolalandro/gimp-baloon-plugin/raw/master/imgs/demo3.png)

* here the results
![result](https://github.com/nicolalandro/gimp-baloon-plugin/raw/master/imgs/demo4.png)
